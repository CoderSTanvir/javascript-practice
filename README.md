# javascript-practice

My beginner JavaScript projects from freeCodeCamp.

## Projects

* **Factorial Calculator** (`build-a-factorial-calculator.js`) - Computes the factorial of an integer using algorithmic math loops.
* **Golf Score Translator** (`build-a-golf-score-translator.js`) - Converts a golfer's strokes relative to par into traditional terminology (e.g., "Birdie", "Eagle").
* **Longest Word Finder App** (`build-a-longest-word-finder-app.js`) - Evaluates sentences to dynamically isolate and measure the length of the longest word.
* **Lunch Picker Program** (`build-a-lunch-picker-program.js`) - A functional script that selects or filters meal choices from a menu array.
* **Record Collection** (`build-a-record-collection.js`) - Manages and updates a multi-layered, nested JSON object representing a music database.
* **String Ending Checker** (`implement-a-string-ending-checker-function.js`) - Validates whether a target string finishes with a specified substring without using native `.endsWith()`.
* **Chunky Monkey Algorithm** (`implement-the-chunky-monkey-algorithm.js`) - Splits a single large array into smaller, nested sub-arrays of a designated length.
* **Mutations Algorithm** (`implement-the-mutations-algorithm.js`) - Checks if all the characters of one string exist inside another string, ignoring case sensitivity.
* **Cargo Manifest Validator** (`lab-cargo-manifest-validator.js`) - Evaluates and verifies logistical shipping strings against custom validation parameters.
* **Profile Lookup** (`lab-profile-lookup.js`) - Queries a contact directory array to safely fetch specific user profile properties while handling missing data flags.
* **Quiz Game** (`lab-quiz-game.js`) - A score-tracking logic engine built to check user inputs against multiple-choice answer values.
* **Truncate a String** (`truncate-a-string.js`) - Measures string lengths and shortens sentences exceeding a limit, appending a clean trailing `...`.
* **String Repeating Function** (`build-a-string-repeating-function.js`) - Replicates a string a given number of times using core iterative logic loops.
* **Missing Letter Detector** (`missing-letter-detector.js`) - A clean algorithm that scans consecutive string sequences to find and return skipped characters in the alphabet.
* **Smart Pantry Restocker** (`smart-pantry-restocker.js`) - An inventory management engine that parses raw shipping data, dynamically identifies duplicates, builds sorting/restocking actions based on current inventory, and categorizes goods into specialized kitchen storage zones.
